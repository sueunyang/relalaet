발표자료 · GROW MY EARTH

탄소 절약 행동으로
지구를 키우는 게임

매일의 작은 친환경 행동을 인증하면 나만의 행성이 성장합니다. HTML · CSS · JS와 Google Sheets · Apps Script로 만든 웹 프로젝트입니다.

01 · 문제 정의

환경 실천, 동기부여가 없으면 오래가지 않는다

실천은 알지만

텀블러 사용, 대중교통 이용 같은 행동은 누구나 알고 있지만 꾸준히 이어가긴 어렵습니다.

변화가 안 보임

내 행동이 실제로 얼마나 도움이 되는지 눈에 보이지 않아 지속할 이유를 찾기 힘듭니다.

02 · 솔루션

행동을 인증하면, 지구가 자란다

오늘의 행동 체크
→
포인트 · CO₂ 절감량 적립
→
행성 단계 성장
→
꾸미기 + 랭킹 경쟁

황폐한 행성 → 바다 회복 → 초록 대륙 → 숲 무성 → 빛나는 생태계, 총 5단계로 성장합니다.

03 · 주요 기능

게임처럼 즐기는 지속가능한 습관

STREAK연속 인증 기록으로 습관 형성 유도
MISSION매일 바뀌는 오늘의 미션 + 보너스 포인트
EVENT특정 날짜엔 포인트 2배 이벤트
BADGE조건 달성 시 수집하는 업적 배지
DECOR해금한 아이템으로 나만의 행성 꾸미기
RANKGoogle Sheets 기반 실시간 리더보드

04 · 기술 스택

가볍고 단순한 구조

Front-end

HTML · CSS · JavaScript (프레임워크 없이 순수 코드로 구현, GitHub Pages 배포)

Data & Automation

Google Sheets를 데이터베이스로, Apps Script를 API 서버로 활용해 리더보드를 자동 갱신

저장 방식

localStorage로 개인 기기에 즉시 저장 + 서버 연동 시 전체 랭킹과 동기화

배포

별도 서버 없이 GitHub Pages로 정적 호스팅, 누구나 링크로 접속 가능

05 · 마무리

작은 행동이 모여 하나의 지구가 됩니다

앞으로 사진 인증, 친구 초대, 학급 단체전 등으로 발전시킬 계획입니다. 지금 바로 나의 지구를 키워보세요.
